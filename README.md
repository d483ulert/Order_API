# OrderAPI 주문처리
<h2> 개발 환경</h2>
#Java11<br>
#Spring Boot Gradle<br>
#Aws EC2 Ubuntu <br>
#Docker 20.10.12 <br>
#Mysql 8.0.26<br>
<br>

<h2>디렉토리 구조</h2> <br>
# bakset  - 장바구니  <br>
# item - 상품 <br>
# order - 주문처리 <br>
# orderhist - 주문이력관리 <br>

<h2> 구현내용 <h2> <br>
# basket : 장바구니 추가, 삭제, 장바구니 추가시 상품 중복체크 <br>
# item : 주문처리 시 재고관리 <br>
# order : 주문, 주문취소 <br>
# orderhist : 주문이력관리 <br>
